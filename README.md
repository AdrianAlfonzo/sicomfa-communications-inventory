# SICOMFA — Communications Inventory & Visualization System

SICOMFA is an operational communications inventory and visualization platform designed to support equipment management, operational tracking, and structured reporting workflows.

The project was built to improve the organization, visualization, and reporting of communications assets through a modern frontend interface integrated with REST APIs and containerized development workflows.

---

## Overview

SICOMFA focuses on operational visibility and structured equipment management through interactive dashboards, reporting tools, and API-driven data handling.

The platform was designed to streamline operational workflows, improve reporting efficiency, and provide a more organized way to manage communications-related information.

---

## Features

- Communications equipment inventory visualization
- REST API integration for operational data handling
- Interactive dashboard interface
- Operational tracking workflows
- Dynamic PDF report generation
- Structured asset organization
- Containerized development workflows using Docker
- Responsive frontend interface
- Modular frontend architecture

---

## Tech Stack

- React
- JavaScript
- REST APIs
- Docker

- AI-assisted development workflows

## Purpose

I built SICOMFA to solve real operational workflow needs related to communications equipment organization, visualization, and reporting.
